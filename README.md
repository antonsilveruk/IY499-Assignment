Program Description
This application is a Student Grade Tracker built using Python and the Tkinter library. It is designed to help users manage academic records through a simple graphical interface. Users can input a student's name and their numerical grade, which is then stored in a list. The application features a Bubble Sort algorithm to organize students from highest to lowest grades and a search function to find specific records. To ensure data persistence, the program reads from and writes to a 'grades.json' file, so no data is lost when the app is closed. This project demonstrates core programming concepts including file I/O, error handling, and data structures.
